# cat
This is a simple libary to extract text from plaintext, .docx, .odt, .pdf and .rtf files.

## Install
```go get -u github.com/lu4p/cat```

## Basic Usage
```golang 
package main
import (
  "fmt"
  "github.com/lu4p/cat"
)

func main(){
  txt, _ := cat.Cat("filename")
  fmt.Println(txt)
}
```
